# リファレンス - ユーザープロフィールページ

---

## 1. ジャーナル

ユーザーがパフォーマーとして投稿したジャーナルを時系列かつクエスト横断的にスライドショーで閲覧できるプレイヤーです。

- 共通
  - ①投稿日時
  - ②リスペクト数: リスペクトが１件以上ある場合表示。
  - ③テキストオーバーレイスイッチ: （画像が添付されている場合表示）本文表示を切り替えて画像だけの表示にする
  - ④本文: ジャーナル本文
  - ⑤画像: ジャーナルに画像が添付されている場合表示。
  - ⑥クエスト: ジャーナルが投稿されたクエスト情報
  - ⑦ページナビゲーター
    - a. 前のページ
    - b. 現在のページ番号
    - c. 全ページ数
    - d. 次のページ

- パフォーマー以外に表示
  - ⑧「リスペクト」ボタン: 現在表示されているジャーナルにリスペクトを追加します。ボタンが赤くなります。もう一度押すとリスペクトを取り下げます。

    ![User Profile Page 002](/user-manual/ja/reference/user-profile-page-002.png)

- パフォーマー自身のみ表示
  - ⑨プルアップメニュー:
    - a. ジャーナルを削除: **ジャーナル削除の確認** ダイアログが開き、「OK」ボタンを押すと現在表示されているジャーナルが削除されます。
  - ⑩ジャーナル投稿: 現在実行中のクエストにジャーナルを投稿します。「クエスト」で投稿先クエストを選択できます。デフォルトで現在表示されているジャーナルのクエストが選択されます。

    ![User Profile Page 003](/user-manual/ja/reference/user-profile-page-003.png)

---

## 2. ブースト

ユーザーがパフォーマーとして受けたブーストをメッセージとともに時系列かつクエスト横断的に一覧します。下までスクロールすると続きを（あれば）ロードします。

「<img src="/user-manual/search-solid.svg" className="inline" height="12" />」を押すと **EXPLORE BOOSTS** を開き、全てをリストすることができます。

![User Profile Page 005](/user-manual/ja/reference/user-profile-page-005.png)

---

## 3. 最近のクエスト

ユーザーがパフォーマーとして実行している・実行したクエストをアクティブな順に最大12件一覧します。

「<img src="/user-manual/search-solid.svg" className="inline" height="12" /> もっと見る」を押すと **EXPLORE QUESTS** で全てをリスト表示することができます。

![User Profile Page 004](/user-manual/ja/reference/user-profile-page-004.png)

---

## 4. オファー

ユーザーがプロモーターとして募集しているオファーを作成順に最大10件一覧します。

「<img src="/user-manual/search-solid.svg" className="inline" height="12" /> もっと見る」を押すと **EXPLORE OFFERS** で全てをリスト表示することができます。

![User Profile Page 006](/user-manual/ja/reference/user-profile-page-006.png)
