# リファレンス - オファーページ

---

## 1. 共通

### 1.1. オファーコンテンツ

- ①タイトル: オファーのタイトル。クエストのタイトルはオファーへ応募する際に設定することができます。
- ②カテゴリ: オファーが推奨するデフォルトのカテゴリです。クエストのカテゴリはオファーへ応募する際、自由に変更することができます。
- ③タグ: オファーが推奨するデフォルトのタグです。クエストのタグはオファーへ応募する際、自由に変更することができます。
- ④受付終了日時: オファーの応募締め切り日時です。締め切りを過ぎるとそれ以上の応募ができなくなりますが、既に実行済みのクエストは続行されます。
- ⑤ステータス
  - 下書き: 応募受付前のオファーです。
  - 受付中: 応募受付中のオファーです。パフォーマーは応募し、クエストを実行開始することができます。
  - 終了: 応募締め切り済のオファーです。パフォーマーは応募、あるいはクエストを実行開始することができません。
- ⑥クエスト最大期間: パフォーマーがクエストの実行に使用できる最大日数です。応募するパフォーマーは規定の日数内でクエストの期限を設定しなければなりません。
- ⑦各種SNSシェアボタン: Twitter, Facebook, Line で **オファーページ** の URL を共有できます。
- ⑧詳細: URL はリンク、「@～」はTwitterプロフィールへのリンクへ、それぞれ自動的に変換されます。
- ⑨マイルストーン: オファーが推奨するデフォルトのマイルストーンです。
- ⑩主催者コメント: オファー主催者より書き込まれたコメントが表示されます。<br />
  コメントは1ページにつき1件ずつ表示され「<img src="/user-manual/chevron-left-solid.svg" className="inline" height="12" />」でより新しいコメント、
  「<img src="/user-manual/chevron-right-solid.svg" className="inline" height="12" />」でより古いコメントへページ送りすることができます。<br />
  デフォルトでは最新のコメントが表示されます。
- ⑪最近のクエスト: 応募して実行されているクエストを最大20件一覧できます。
  「<img src="/user-manual/search-solid.svg" className="inline" height="12" /> もっと見る」を押すと **EXPLORE QUESTS** で全てを一覧することができます。

  ![Offer Page 005](/user-manual/ja/reference/offer-page-005.png)

### 1.2. オファー主催者・統計情報

- ①主催者
- ②オファープロパティ
  - a. 予算: ブーストに掛けられる全体予算
  - b. ブースト額: １回のブースト額
  - c. 実行回数: 実行された回数（中止・キャンセルを除く）
  - d. 実行中: 現在実行中のクエスト数
  - e. 承認クエスト数: 承認されたクエスト数
  - f. 全クエスト数: 中止・キャンセルを含むクエスト数
  - g. リトライ可能: 承認されるまで何度もリトライできるかどうか

## 2. 主催者向け

### 2.1. ボタン

- ①「編集」ボタン: **EDIT OFFER** を開きます。オファーが「下書き」の場合のみ表示され押すことができます。
- ②「募集開始」ボタン: オファーを「受付中」にします。オファーが「下書き」の場合のみ表示され押すことができます。
- ③「タイトル・カテゴリ・タグ編集」ボタン: タイトル、カテゴリ、タグを編集するフォームを開きます。
- ④「コメントを投稿」ボタン: 主催者コメントを投稿するフォームを開きます。
- ⑤「削除」ボタン: オファーを削除します。オファーが「下書き」の場合のみ表示され押すことができます。
- ⑥「募集終了」ボタン: オファーを「終了」にします。オファーが「受付中」の場合のみ表示され押すことができます。

## 3. パフォーマー向け

### 2.2. ボタン

- ①「応募」ボタン: 応募前のオファーに表示されます。ボタンを押すと **CREATE NEW QUEST** を開き、オファーに応募するクエストを作成できます。
- ②「応募済み」ボタン: 応募済みのオファーに表示されます。ボタンを押すと **クエストページ** を開きます。
