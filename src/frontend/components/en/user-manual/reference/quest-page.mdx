# リファレンス - クエストページ

---

## 1. 共通

### 1.1. ジャーナル

パフォーマーが投稿したジャーナルをスライドショーで閲覧できるプレイヤーです。

- ①投稿日時
- ②リスペクト数: リスペクトが１件以上ある場合表示。
- ③テキストオーバーレイスイッチ: （画像が添付されている場合表示）本文表示を切り替えて画像だけの表示にする
- ④本文: ジャーナル本文
- ⑤ページナビゲーター
  - a. 前のページ
  - b. 現在のページ番号
  - c. 全ページ数
  - d. 次のページ
- ⑥画像: ジャーナルに画像が添付されている場合表示。

  ![Quest Page 004](/user-manual/ja/reference/quest-page-004.png)

### 1.2. ブースト

クエストに対して行われたブーストをメッセージとともに時系列で一覧します。下までスクロールすると続きを（あれば）ロードします。

「<img src="/user-manual/search-solid.svg" className="inline" height="12" />」を押すと **EXPLORE BOOSTS** を開き、全てをリストすることができます。

![Quest Page 005](/user-manual/ja/reference/quest-page-005.png)

### 1.3. クエストコンテンツ

- ①ステータス
  - PENDING: 保留中
  - PRE PERFORMANCE: 実行前
  - CANCELED: キャンセル
  - PERFORMING: 実行中
  - ABORTED: 中止
  - VOTING: 投票中
  - FINISHED: 終了
  - APPROVED: 承認
- ②タイトル
- ③カテゴリ: カテゴリを押すと **EXPLORE QUESTS** を開く。
- ④タグ: タグを押すとすると **EXPLORE QUESTS** を開く。
- ⑤期限: クエストの完了期限
- ⑥進捗度（％）
- ⑦各種SNSシェアボタン: Twitter, Facebook, Line で **クエストページ** の URL を共有できます。
- ⑧詳細: URL はリンク、「@～」はTwitterプロフィールへのリンクへ、それぞれ自動的に変換されます。
- ⑨マイルストーン: 完了している物にはチェックが入ります。
- ⑩前回のクエスト: コンティニューしたクエストの場合に表示されます。
- ⑪ディスカッション
  - a. 投稿者: アバター、あるいはリンクを押すと **ユーザープロフィール** を開きます
  - b. 投稿日時: 現時刻から遡る時間で表示されます
  - c. 引用: 投稿が他投稿への返信の場合表示。
  - d. 投稿本文: URL はリンク、「@～」はTwitterプロフィールへのリンクへ、それぞれ自動的に変換されます。
  - e. 返信: 投稿に返信する形で **新規コメント投稿** を開きます
  - f. 削除: （投稿者、あるいはパフォーマーのみ表示）**ディスカッション削除の確認** ダイアログが開いた後、「OK」ボタンを押すと該当の投稿を削除できます。
    パフォーマーは自身のクエストのすべてのディスカッション投稿を削除できます。
    尚、投稿内容は削除されますが投稿自体は残ります（「この投稿は削除されました」と表示）
  - g. ディスカッション投稿: **新規コメント投稿** を開きます

  ![Quest Page 003](/user-manual/ja/reference/quest-page-003.png)

### 1.4. パフォーマー・統計情報

- ①パフォーマー: 「アバター」を押すと **ユーザープロフィール** を開きます
- ②金庫
  - a. Type A（緑）: ブースト戦略「A. 実行者を信頼」でブーストされた割合（％）
  - b. Type B（青）: ブースト戦略「B. 投票結果を信頼」でブーストされた割合（％）
  - c. Type C（黄）: ブースト戦略「C. 最終判断が必要」でブーストされた割合（％）
  - d. 合計金額（XYM）
- ③ブースト統計情報
  - e. ブースト回数
  - f. 授与: パフォーマーに授与された金額
  - g. 返金: クライアントに返金された金額
  - h. 最大容量: 金庫に貯め込める最大の合計金額
- ④投票（投票開始後表示）
  - i. Positive（緑）: 承認に投票された割合（％）
  - j. Negative（黄）: 否認に投票された割合（％）
  - k. 合計投票数
- ⑤投票統計（投票開始後表示）
  - l. あなたの票: 投票している場合は「承認」または「否認」、投票していなければ「未投票」
  - m. 投票締め切り: 投票が締め切られる日時
  - n. 投票結果: （投票締め切り後表示）「承認」または「否認」

  ![Quest Page 006](/user-manual/ja/reference/quest-page-006.png)

## 2. パフォーマー向け

### 2.1. ボタン

- PRE PERFORMANCE
  - ①「編集」ボタン: （実行前のみ表示）**EDIT QUEST** を開きます。
  - ②「実行」ボタン: （実行前のみ表示）**実行の確認** ダイアログが開き、「OK」ボタンを押すとクエストを実行中にします。
  - ③「キャンセル」ボタン: （実行前のみ表示）**キャンセルの確認** ダイアログが開き、「OK」ボタンを押すとクエストを実行前にキャンセルにします。

    ![Quest Page 001](/user-manual/ja/reference/quest-page-001.png)

- PERFORMING
  - ④「クエストを完了」ボタン: （実行中のみ表示）**完了の確認** ダイアログが開き、「OK」ボタンを押すとクエストを完了して、投票中にします。
  - ⑤「中止」ボタン: （実行中のみ表示）**中止の確認** ダイアログが開き、「OK」ボタンを押すとクエストを中止にします。

    ![Quest Page 002](/user-manual/ja/reference/quest-page-002.png)

- その他
  - ⑥「コンティニュー」ボタン: （キャンセル・中止・投票中・終了・承認において表示）**CREATE NEW QUEST** をクエストの内容が引き継がれた状態で開きます。

    ![Quest Page 009](/user-manual/ja/reference/quest-page-009.png)

### 2.2. ジャーナル

- ①プルアップメニュー（パフォーマーの場合のみ表示）
  - a. ジャーナルを削除: **ジャーナル削除の確認** ダイアログが開き、「OK」ボタンを押すと現在表示されているジャーナルが削除されます。
- ②ジャーナル投稿（パフォーマーの場合のみ表示）: クエストにジャーナルを投稿します。<br />
  ※一度投稿したジャーナルは編集することはできません。（削除のみ）

  ![Quest Page 007](/user-manual/ja/reference/quest-page-007.png)

### 2.3. クエストコンテンツ

- ①メタデータ編集: タイトル・カテゴリ・タグ・言語・金庫の最大容量を編集するフォームを開きます。
- ②進捗度変更: （マイルストーンが一つもない場合表示）進捗度（％）を変更するフォームを開きます。
- ③マイルストーン編集: マイルストーンを編集・追加・削除・順序変更するフォームを開きます。

  ![Quest Page 008](/user-manual/ja/reference/quest-page-008.png)

## 3. クライアント向け

### 3.1. ボタン

- 共通
  - ①「購読」ボタン: クエストを購読します。
    購読中の場合は「購読中 <img src="/user-manual/times-circle-solid.svg" className="inline" height="12" />」ボタンに変わり、押すことで購読を解除できます。
- PERFORMING
  - ②「ブースト」ボタン: **BOOST** フォームを開きます。10件以上の未払いのブースト請求書がある場合、グレーアウトして押せなくなります。
  - ③「承認に期日前投票」ボタン: （ブーストしたクエストで表示）投票開始前に、承認に期日前投票します。投票締め切りまで何度でも変更することができます。
  - ④「否認に期日前投票」ボタン: （ブーストしたクエストで表示）投票開始前に、否認に期日前投票します。投票締め切りまで何度でも変更することができます。

    ![Quest Page 010](/user-manual/ja/reference/quest-page-010.png)

- VOTING
  - ⑤「ブースト」ボタン: 投票中のクエストでもブーストする事が可能です。
  - ⑥「承認に投票」ボタン: （投票中のみ表示）投票締め切りまで何度でも変更することができます。
  - ⑦「否認に投票」ボタン: （投票中のみ表示）投票締め切りまで何度でも変更することができます。

    ![Quest Page 011](/user-manual/ja/reference/quest-page-011.png)

- APPROVED
  - ⑧「授与する」ボタン: （Type.Cブーストしたクエストで表示）パフォーマーにブーストを授与します。一度授与したら戻すことはできません。
  - ⑨「返金を受ける」ボタン: （Type.Cブーストしたクエストで表示）ブーストしたXYMを返金してもらいます。一度返金したら戻すことはできません。

    ![Quest Page 012](/user-manual/ja/reference/quest-page-012.png)

### 3.2. ジャーナル

- ①「リスペクト」ボタン: 表示されているジャーナルにリスペクトを追加します。ボタンが赤くなります。もう一度押すとリスペクトを取り下げます。

  ![Quest Page 013](/user-manual/ja/reference/quest-page-013.png)

  ![Quest Page 014](/user-manual/ja/reference/quest-page-014.png)

## 4. リクエスト

- ①依頼者: リクエストの送信元です。「アバター」を押すと **ユーザープロフィール** を開きます。

  ![Quest Page 015](/user-manual/ja/reference/quest-page-015.png)

## 5. オファー

- ①オファー: クエストが応募しているオファーです。「アバター」を押すと **オファーページ** を開きます。

  ![Quest Page 016](/user-manual/ja/reference/quest-page-016.png)
