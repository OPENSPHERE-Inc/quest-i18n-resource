# ガイド - クエスト実行

---

## 1. 概要

クエスト実行の基本的な（推奨する）サイクルは以下の通りです。

1. **クエストページ** で「実行」ボタンを押す（クエスト開始）
2. SNS 等でフォロワーに共有する（任意）
3. クエスト活動する。
4. マイルストーンの達成した項目にチェックを入れる。<br />
   （マイルストーンがない場合は）スライダーで進捗度を増加させる。
5. ジャーナルに進捗やクエストでの出来事を投稿する。<br />
   ジャーナル更新を SNS 等で共有する（任意）
6. 進捗が 100% になるまで 2 ～ 5 を繰り返す。
7. 期限までにクエストページで「完了」ボタンを押す。<br />
   完了したことを SNS 等で共有する（任意）
8. 投票結果を待つ（24時間）
9. （結果次第）金庫の XYM が手に入る。<br />
   ジャーナル更新はクエスト終了後も可能です（任意）
10. 活動を続行する為にコンテニューする（任意）<br />
    1 に戻って繰り返し。

## 2. 進捗度更新

### 2.1. マイルストーンの設定がある場合

マイルストーンとは、クエストの進行を番号付き箇条書きでまとめたものです。

項目毎にチェックボックスが用意されており、達成するごとにチェックを入れることで、進捗度（％）が更新されます。

![Quest Performance 001](/user-manual/ja/guide/quest-performance-001.png)

マイルストーンはクエスト実行中でも自由に項目を増減させたり、内容を変更したり、順番を入れ替えられます。
従って、ゴールだけ決めて途中の予定が全く決まっていないクエストでも、後からマイルストーンを追加していくことができます。

マイルストーンの右横の「<img src="/user-manual/edit-regular.svg" className="inline" height="12" />」を押して、
以下の様なマイルストーン編集フォームを開いてください。

![Quest Performance 002](/user-manual/ja/guide/quest-performance-002.png)

マイルストーンをマメに進行させることで、サポーターにクエストの活動がアクティブであることをアピールしましょう。

### 2.2. マイルストーンの設定がない場合

クエストにマイルストーンを設定することは任意になります。
設定がない場合は、スライダーで進捗度（％）を変更できます。

1. 実行中の **クエストページ** で **プログレスバー** の右横の「<img src="/user-manual/edit-regular.svg" className="inline" height="12" />」を押してください。

   ![Quest Performance 003](/user-manual/ja/guide/quest-performance-003.png)

2. スライダーで進行度を変更してください。左いっぱいで 0%、右いっぱいで 100% です。<br />
   「オンスケ」ボタンを押すと、期限に対する残り日数で進行度（％）を計算してくれます。
3. 「保存」ボタンを押してください。

   ![Quest Performance 004](/user-manual/ja/guide/quest-performance-004.png)

## 3. ジャーナルの投稿

### 3.1. ジャーナルとは

ジャーナルとは、クエストの日誌です。

短いテキストで進捗をお知らせしたり、スクリーンショットや写真で成果を披露したり、動画を埋め込んだり、配信を埋め込んだりすることができます。
配信視聴ページを意識した設計になっており、表示領域が 16:9 に固定されるので、長文を読ませるのにはそれほど向いてません。
従って、外部ブログサイトへリンクを張るなどして誘導することも可能です。

![Quest Performance 007](/user-manual/ja/guide/quest-performance-007.png)

ジャーナルは **BASECAMP** と、フォロワーや購読者の **DASHBOARD** に新着順に表示されますから、
こまめに更新することでクエストを盛り上げる効果が見込めるでしょう。

![Quest Performance 005](/user-manual/ja/guide/quest-performance-005.png)

![Quest Performance 006](/user-manual/ja/guide/quest-performance-006.png)

また、ジャーナルの投稿は実行前、実行中、終了後、中止後等、いずれの状態であっても行うことができます。
つまり、終了したクエストでブーストへお礼や、クエスト後の結果報告、次回の見通しなどを書き込むことができます。

### 3.2. 投稿の手順

投稿は以下のように行います。

1. **クエストページ** でジャーナルの「<img src="/user-manual/plus-solid.svg" className="inline" height="12" />」を押してください。

   ![Quest Performance 008](/user-manual/ja/guide/quest-performance-008.png)

2. **新規ジャーナル投稿** フォームが表示されるので、以下のように入力してください。
   - ①クエスト: ジャーナルを投稿するクエストが選択されているか確認してください（デフォルトで選択されています）
   - ②クエストで何が起こっているか書き出す: ジャーナルの内容を書いてください。
   - ③画像ファイルを選択: ジャーナルに画像を添付したい場合は、ここでローカルドライブ内のファイルを選択してください。<br />
     画像は、ジャーナル1件につき1枚まで添付可能です。
3. 以下の隠し項目は、通常は触る必要がありません。
   - ④言語: ジャーナルがどの言語で書かれているかを設定してください。デフォルトでユーザープロフィールの言語が選択されています。
4. 最後に「保存」ボタンを押してください。<br />
   画像を添付する場合はサーバーでの処理（リサイズやサムネイル生成）に少しだけ時間がかかります。

   ![Quest Performance 009](/user-manual/ja/guide/quest-performance-009.png)

> #### 画像の変換処理について
>
> アップロードした画像は必ず 16:9 になるようにリサイズされます。16:9 でない画像はリサイズの際に余白が付与されます。
> この余白は元画像を拡大してぼかすという方法で自動生成されます。

### 3.3. ジャーナルの削除

誤って投稿したジャーナルは、以下のように削除することができます。
アップロードした画像も併せて削除されます。

1. 削除したいジャーナルを開いた状態で、左下の「<img src="/user-manual/ellipsis-v-solid.svg" className="inline" height="12" />」を押してください。
2. メニューから「ジャーナルを削除」を押してください。

   ![Quest Performance 010](/user-manual/ja/guide/quest-performance-010.png)

3. **ジャーナル削除の確認** ダイアログが開きますので、このまま削除してよければ「OK」ボタンを押してください。

   ![Quest Performance 011](/user-manual/ja/guide/quest-performance-011.png)

## 4. ディスカッションの活用

### 4.1. ディスカッションとは

ディスカッションは、クエスト毎に用意されている掲示板機能です。
これによって、ブースト以外でパフォーマーとサポーターがコミュニケーションを取ることができます。

ディスカッションは、QUEST にユーザー登録しサインインさえしていれば、誰でも投稿することが可能です。

![Quest Performance 012](/user-manual/ja/guide/quest-performance-012.png)

### 4.2. 投稿の手順

新たな投稿は以下のように行います。

1. **クエストページ** を「ディスカッション」までスクロールし、「ディスカッションを投稿」を押してください。

   ![Quest Performance 014](/user-manual/ja/guide/quest-performance-014.png)

2. **ディスカッション** フォームが開きますので、「新規コメント投稿」にコメントを入力してください。<br />
   隠し項目「言語」はコメントがどの言語で書かれているかの設定で、通常は触る必要はありません。デフォルトでユーザープロフィールの言語が選択されています。
3. 「保存」ボタンを押してください。コメントがディスカッションに投稿されます。

   ![Quest Performance 013](/user-manual/ja/guide/quest-performance-013.png)

また、コメントに対して以下のように「返信」を行うことも可能です。

1. **ディスカッション** に投稿されている任意のコメントで、「<img src="/user-manual/reply-solid.svg" className="inline" height="12" /> 返信」を押してください。自分の投稿にも返信可能です。

   ![Quest Performance 015](/user-manual/ja/guide/quest-performance-015.png)

2. 該当のコメントが引用された状態で **ディスカッション** フォームが開きますので、「新規コメント投稿」にコメントを入力してください。<br />
   隠し項目「言語」はコメントがどの言語で書かれているかの設定で、通常は触る必要はありません。デフォルトでユーザープロフィールの言語が選択されています。
3. 「保存」ボタンを押してください。コメントが引用付きでディスカッションに投稿されます。

   ![Quest Performance 016](/user-manual/ja/guide/quest-performance-016.png)

### 4.3. コメントの削除

ディスカッションへ投稿したコメントはいつでも削除することが可能です。**尚、一度削除したコメントは元に戻すことができません**

1. **ディスカッション** に自分が投稿したコメントで、「<img src="/user-manual/trash-alt-solid.svg" className="inline" height="12" /> 削除」を押してください。
   クエストのパフォーマーの場合、全てのコメントが削除可能です。

   ![Quest Performance 017](/user-manual/ja/guide/quest-performance-017.png)

2. **ディスカッション削除の確認** ダイアログが開きますので、このまま削除してよければ「OK」ボタンを押してください。

   ![Quest Performance 018](/user-manual/ja/guide/quest-performance-018.png)

3. コメントの本文が **この投稿は削除されました。** という表示に変わります。

   ![Quest Performance 023](/user-manual/ja/guide/quest-performance-023.png)

### 4.4. コメントの非表示

自分がパフォーマーのクエストでは、ディスカッションへの投稿を管理することができます。

以下の様に、荒らしコメントや、不適切なコメントを非表示にできます。非表示状態はいつでも解除することができます。

1. **ディスカッション** に自分以外が投稿したコメントで、「非表示にする」を押してください。

   ![Quest Performance 019](/user-manual/ja/guide/quest-performance-019.png)

2. **ディスカッション非表示の確認** ダイアログが開きますので、このまま非表示にしてよければ「OK」ボタンを押してください。

   ![Quest Performance 020](/user-manual/ja/guide/quest-performance-020.png)

3. コメントの本文が **この投稿は実行者によって非表示にされています。** という表示に変わります。
4. 再び表示状態にするには「<img src="/user-manual/eye-solid.svg" className="inline" height="12" /> 表示する」を押してください。

   ![Quest Performance 021](/user-manual/ja/guide/quest-performance-021.png)

5. **ディスカッション表示の確認** ダイアログが開きますので、このまま表示してよければ「OK」ボタンをおしてください。

   ![Quest Performance 022](/user-manual/ja/guide/quest-performance-022.png)

6. コメントの本文が表示状態に戻ります。

## 5. クエストの完了

進捗度が 100% かどうかによらず、期限までに **クエストページ** の「完了」ボタンを押すと、投票フェーズに移行します。
**完了の確認** ダイアログが開きますので、このまま完了してよければ「OK」ボタンを押してください。

あるいは、進捗度が 100% になっている場合、期限を過ぎ多としても自動的に投票フェーズに移行します。

![Quest Performance 024](/user-manual/ja/guide/quest-performance-024.png)

![Quest Performance 025](/user-manual/ja/guide/quest-performance-025.png)

## 6. クエストの中止

何らかの理由でクエストを中止したくなった場合は **クエストページ** で「中止」ボタンを押してください。
**中止の確認** ダイアログが開きますので、このまま中止してよければ「OK」ボタンを押してください。

また、クエストの進捗度が 100% 未満の状態で、かつ「完了」ボタンを押さないまま期限を過ぎると、そのクエストは中止されたものとして扱われます。

クエストが中止されると、金庫の XYM は全てサポーターに返金されます。

![Quest Performance 026](/user-manual/ja/guide/quest-performance-026.png)

![Quest Performance 027](/user-manual/ja/guide/quest-performance-027.png)
