## 一般的な用語

| 用語              | 意味                                     |
|-----------------|----------------------------------------|
| ブースト            | 投げ銭                                    |
| クエスト            | 投げ銭対象のプロジェクト                           |
| パフォーマー（実行者）     | クエストを実行する人                             |
| サポーター           | クエストをブーストする人                           |
| クライアント（依頼者）     | リクエストを作成する人                            |
| プロモーター（主催者）     | オファーを主催する人                             |
| フォロワー           | ユーザーを（QUEST や SNS で）フォローする人、クエストを購読する人 |
| ユーザー            | QUEST ユーザー全体                           |
| 仮想通貨            | 暗号通貨、暗号資産とも。ブロックチェーン上で取引するお金。          |
| ブロックチェーン        | 仮想通貨の取引を記録する分散型台帳                      |
| Symbol ブロックチェーン | 数あるブロックチェーンプロジェクトの一つ                   |
| XYM（ジム）         | Symbol ブロックチェーン上で取引される仮想通貨（の単位）        |

## クエストステータス

| ステータス名 | クエストの状態                            | 　英語表記           |
|--------|------------------------------------|-----------------|
| 保留中    | クエストの作成が保留されている（手数料未払い等）           | PENDING         |
| 実行前    | クエストが作成され、未だ実行されていない               | PRE PERFORMANCE |
| キャンセル  | クエストが実行に移されずにキャンセルされた              | CANCELED        |
| 実行中    | クエストが今現在実行中である                     | PERFORMING      |
| 中止     | クエストの実行がパフォーマーにより中止された、または期限切れになった | ABORTED         |
| 投票中    | クエストが完了し、投票期間中である                  | VOTING          |
| 終了     | クエスト（の投票期間）が未承認で終了                 | FINISHED        |
| 承認     | クエストが投票により承認された                    | APPROVED        |
