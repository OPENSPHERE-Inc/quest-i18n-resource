# リファレンス - 探す

---

## 1. クエスト

- ①「クエストを作成」ボタン: **CREATE NEW QUEST** を開きます。
- ②カテゴリ: 検索結果を選択したカテゴリに限定します。
  - 全て: カテゴリを限定しません（デフォルトで選択）
  - ゲーミング: ゲーム配信、e-Sports大会、RTA等
  - テクノロジー: アプリ開発、ハードウェア開発、技術研究や学習等
  - クリエイティブ: イラスト制作、作詞作曲活動、アート活動、DIY等
  - アウトドア: アウトドアアクティビティ、釣り、登山等
  - スポーツ: 大会参加、筋トレ、フィットネス等
  - カーライフ: ツーリング、カスタム、競技車製作、スポーツ走行
  - 生活: 料理、掃除、買い物等
- ③言語: 検索結果を選択した言語に限定します。
  - 全言語: 全ての言語のコンテンツを表示します。
  - 自分の言語: プロフィールで設定した言語のコンテンツに限定します。
- ④タグ: 検索結果をタグで絞込します。カンマ区切りで複数指定でき、その場合 OR 検索となります。大文字小文字の区別はしません。
- ⑤ステータス: 検索結果をステータスで絞り込みします。
  - 全て: ステータスの絞り込みをしません（デフォルトで選択）
  - 保留中: クエストの作成が保留されている
  - 実行前: クエストが作成され、未だ実行されていない
  - キャンセル: クエストが実行前にキャンセルされた
  - 実行中: クエストが実行中である
  - 中止: クエストの実行がパフォーマーにより中止された、または期限切れになった
  - 投票中: クエストが完了し、投票期間中である
  - 終了: クエスト（の投票期間）が未承認で終了
  - 承認: クエストが投票により承認された
- ⑥ソート: 検索結果を並べ替えします。
  - より最近状態が変化: クエストのアクティブさの降順でソート
  - より新しい: クエストの作成日時の降順でソート
  - より古い: クエスト作成日時の昇順でソート
  - より最近更新: クエスト更新日時の降順でソート

  ![Explore 001](/user-manual/ja/reference/explore-001.png)

検索結果の行を選択すると、右側に **クエストサマリ** が表示されます。

- ⑦「閉じる」ボタン: **クエストサマリ** を閉じます
- ⑧「ブースト」ボタン: **BOOST** フォームを開きます
- ⑨「詳細」ボタン: **クエストページ** を開きます

  ![Explore 002](/user-manual/ja/reference/explore-002.png)

検索結果の行を選択すると、右側に **クエストサマリ** が表示されます。

- ⑤「閉じる」ボタン: **クエストサマリ** を閉じます。

  ![Explore 006](/user-manual/ja/reference/explore-006.png)

**クエストサマリ** は **ユーザーサマリ** の表示に切り替えができます。

- ⑩「フォロー」ボタン: ユーザーをフォローします。
  既にフォローしている場合は「フォロー中 <img src="/user-manual/times-circle-solid.svg" className="inline" height="12" />」ボタンが表示され、押すとフォローを解除します。
- ⑪「リクエスト」ボタン: **CREATE NEW REQUEST** フォームを開きます。
- ⑫「詳細」ボタン: **ユーザープロフィール** を開きます。
- ⑬クエストサマリ: **ユーザーサマリ** を閉じて **クエストサマリ** に切り替えます。

  ![Explore 007](/user-manual/ja/reference/explore-007.png)

## 4. ユーザー

- ①カテゴリ: 検索結果を選択したカテゴリに限定します（実行中のクエストのカテゴリが対象です）
  - 全て: カテゴリを限定しません（デフォルトで選択）
  - ゲーミング: ゲーム配信、e-Sports大会、RTA等
  - テクノロジー: アプリ開発、ハードウェア開発、技術研究や学習等
  - クリエイティブ: イラスト制作、作詞作曲活動、アート活動、DIY等
  - アウトドア: アウトドアアクティビティ、釣り、登山等
  - スポーツ: 大会参加、筋トレ、フィットネス等
  - カーライフ: ツーリング、カスタム、競技車製作、スポーツ走行
  - 生活: 料理、掃除、買い物等
- ②言語: 検索結果を選択した言語に限定します（プロフィールの言語が対象です）
  - 全言語: 全ての言語のコンテンツを表示します
  - 自分の言語: プロフィールで設定した言語のコンテンツに限定します
- ③名前: ユーザー名でフィルターします。前方一致で、大文字小文字を区別します。名前でフィルターする場合、ソートが「名前昇順」に変更されます。
- ④ソート: 検索結果を並べ替えします。
  - より新しい: ユーザー登録日時の降順でソート
  - より古い: ユーザー登録日時の昇順でソート
  - より最近クエスト実行: パフォーマーとしてクエスト実行日時の降順でソート。尚、クエストを実行していないと検索結果に含まれません
  - より最近更新: ユーザー更新日時の降順でソート
  - 名前昇順: ユーザー名の昇順でソート
  - 名前降順: ユーザー名の降順でソート

  ![Explore 008](/user-manual/ja/reference/explore-008.png)

検索結果の行を選択すると、右側に **ユーザーサマリ** が表示されます。

- ⑤「閉じる」ボタン: **ユーザーサマリ** を閉じます。
- ⑥「フォロー」ボタン: ユーザーをフォローします。
  既にフォローしている場合は「フォロー中 <img src="/user-manual/times-circle-solid.svg" className="inline" height="12" />」ボタンが表示され、押すとフォローを解除します。
- ⑦「リクエスト」ボタン: **CREATE NEW REQUEST** フォームを開きます。
- ⑧「詳細」ボタン: **ユーザープロフィール** を開きます。

  ![Explore 009](/user-manual/ja/reference/explore-009.png)
